# Book Recommender Model 

--- 

Tensorflow model for recommending books based on your (and others) previously read books.  
Accuracy of recommendation scaled to number of reccomendations:
* 1 - 0.36%
* 5 - 0.42%
* 10 - 0.55%
* 50 - 0.18%
* 100 - 0.12%
